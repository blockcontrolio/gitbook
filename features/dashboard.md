# Dashboard

The BlockControl Dashboard provides a real-time overview of your counterparty's assets, accounts, and recent activity in a single aggregated view.

---

## Dashboard Components

### Asset Summary
- Total token supply per asset
- Number of active tokens
- Quick link to token management

### Account Overview
- Total accounts by type (Admin, Issuer, Operator, Distributor, Client)
- Enabled vs. disabled account count
- Quick link to account management

### Recent Transfers
- Last 10 transfers with status indicators
- Filter by direction (Incoming / Outgoing)
- Click any transfer to open the detail page

### Balance Snapshot
- Aggregated balances across all accounts, grouped by token
- Real-time sync with on-chain state

### Pending Operations
- Transactions currently in a PENDING state awaiting blockchain confirmation
- Live status updates

---

## Navigation

The dashboard acts as the central hub of the platform. All major sections are accessible from the sidebar:

- **Assets** — Token management, minting, burning
- **Accounts** — Account provisioning and role management
- **Transfers** — Transfer history and initiation
- **Invoices** — Invoice creation and management
- **Partnerships** — Partner counterparty management
- **Settings** — User management, authentication configuration

---

## Cross-Links

Each item in the dashboard is clickable and navigates to its dedicated detail page — whether a token, account, or transaction. This makes it easy to drill down from a high-level overview into full transaction or account details without navigating through menus manually.
